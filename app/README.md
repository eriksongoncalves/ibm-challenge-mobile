## Configurações

<br />

Na raiz do projeto instale as dependências:

```
npm install
```

<br />

## Iniciando a aplicação:

<br />

Renomeie o arquivo `.env.example` na raiz do projeto para `.env` e altere os valores

<br />

Execute a aplicação e rode o comando no terminal.

```
npm run android
npm run ios
```
